# Leave System BE

This project is a simple leave system developed with Spring Boot.

## Getting Started

### Prerequisites
- Java 17
- Maven
- Docker

### Installation
1.  Clone this repository.
2.  Start Docker and run the command `docker-compose up -d` to start the PostgreSQL database.
3.  Run the Spring Boot application with the command `mvn spring-boot:run`.
4.  The API will be available at `http://localhost:8080`.

## API Documentation

You can view the API documentation and test the endpoints at Swagger UI:
[http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
